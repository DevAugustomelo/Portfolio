Tecnologias

HTML

Hypertext Markup Language, um sistema padronizado para marcar arquivos de texto. HTML5 é uma linguagem de marcação de tecnologia central da Internet usada para estruturar e apresentar conteúdo para a World Wide Web.

Informações:
História: wikipedia.org
Site Oficial: w3.org
Vantagens:
  • Fácil de usar
  • Fácil de codificar
  • Permite uso de Modelos e Integrações
Desvantagens:
  • Precisa escrever muito código para criar uma página da web simples
  • Não pode produzir saída dinâmica sozinho
  • Os recursos de segurança oferecidos pelo HTML são limitados
CSS

Cascading Style Sheets (CSS) é um mecanismo simples para adicionar estilo (por exemplo, fontes, cores, espaçamento) a documentos da Web.

Informações:
História: wikipedia.org
Site Oficial: w3.org
Vantagens:
  • Com CSS economiza-se tempo de criação e manutenção do site (o código de formatação fica isolado em um único arquivo CSS)
  • Redução do código HTML
  • Maior eficiência no gerenciamento do Layout
Desvantagens:
  • As funcionalidades do CSS não são compatíveis com todos os navegadores
  • Antes de implementar algo precisa checar navegadores compatíveis
  • Manuseio complexo para iniciantes
Java Script

É uma linguagem de programação conhecido como a linguagem de script para páginas da Web. Permite implementar itens complexos fanzendo mais que simplesmente mostrar informação estática.

Informações:
História: devmedia.com
Site Oficial: nodejs
Vantagens:
  • Versatilidade da linguagem
  • Ótima linguagem para iniciantes em programação
  • Compatível com uma grande variedade de navegadores e plataformas
Desvantagens:
  • Poucos recursos voltados à segurança
  • A renderização pode causar problemas de desempenho
  • Pode ser usada para executar programas maliciosos sem que o usuário saiba
